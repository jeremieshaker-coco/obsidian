# Task Service gRPC Endpoints

This document lists the gRPC endpoints for the **[[Task Service]]**.

*   **`Create`**: Creates a new [[Task]].
*   **`Get`**: Retrieves a [[Task]] by its ID.
*   **`GetHistory`**: Gets the event history for a specific [[Task]].
*   **`Cancel`**: Cancels an existing [[Task]].
*   **`GetProviders`**: Retrieves a list of available providers (e.g., [[Robot]]s, [[FO (Field Operator)|Field Ops]]) that can perform a certain [[Task]].
*   **`GetProviderHistory`**: Gets the [[Task]] history for a specific provider.
